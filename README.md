#TheDogsPage

HTML study page created for practice:

- Semantic structure (`<header>`, `<main>`, `<footer>`, `<h2>`, `<ul>`, etc.)
- Form with radio buttons and checkboxes
- Image insertion with alt attribute
- External links

## Images
The images are in the `images/` folder and used in the HTML with relative paths.
